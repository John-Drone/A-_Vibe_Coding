# A*_Vibe_Coding
新手向，一个快速上手的A*、JMP算法自学项目，包含测试、数据对比功能，四种地图供数据测试。
运行： .\pathfinding.exe
# 代码内含7个思考题
### 1. 对角线移动的角落切割检查（grid_map.cpp:78-82） 当从 (x,y) 对角移动到 (x+dx, y+dy) 时，需要检查 (x+dx, y) 和 (x, y+dy) 是否都是可通行的，防止"穿墙角"。
### 2. A* 中 Open List 的重复节点处理（a_star.cpp:68-70） 为什么 priority_queue 不支持修改？用"懒删除"策略（push 新的，取出时跳过 Closed 节点）有什么优缺点？
### 3. A* 中更新邻居的条件判断（a_star.cpp:82-83） tentative_g < neighbor->g vs tentative_g <= neighbor->g ，哪种正确？为什么？
### 4. JPS 对角线跳跃时为什么要检查水平和垂直方向（jps.cpp:88-92） 如果水平方向存在跳点，当前点为什么也是跳点？这和"被迫邻居"的概念有什么关系？
### 5. JPS 非起点方向剪枝（jps.cpp:172-192） 水平/垂直移动时，什么情况下需要添加对角线方向？这和被迫邻居的检测有什么联系？
### 6. JPS 路径插值（jps.cpp:210-214） JPS 返回的是跳点序列，如何在相邻跳点之间填充完整的逐格路径？
### 7. JPS 代价计算（jps.cpp:52-55） 跳点之间可能跨越多个格子，用启发函数近似是否精确？更正确的做法是什么？
